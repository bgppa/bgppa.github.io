Belts represent proficienty level. Ispired from Brasilian Jiu Jitsu.

Belt color Symbol x = % Completion
White 1 < x < 35
Blue 34 < x < 70
Purple 69 < x < 100
Brown x = 100

For the BLACK belt, each game has its own particular challenge.

Pokemon Violet

% given by the number of catched Pokemon. Currently 155 (total is 400). Black belt: tbd.

Pokemon Arceus

Completion = rank as Star Member. 7,8th rank blue, 9,10th = brown,.... Black Belt: complete all the secondary missions.

Sekiro

Black Belt: Complete the Mortal Journey.

Metal Gear Solid 5 GZ

Black Belt: 100% completion rate in-game. DONE!

Ninja Gaiden DS

Number of talismans (tot is 45, current is 40), therefore brown belt is ok. All 45 give the Black Belt direcly.
