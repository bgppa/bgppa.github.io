Belts represent proficiency levels. Ispired from Jiu Jitsu.

Belt color Symbol x = % Completion
White 1 < x < 25
Blue 25 <= x < 50
Purple 50 <= x < 75
Brown 75 <= x < 100
Black x = 100
Red specific
